Discord bot coded in javascript made using discord.js. Has the following commands:

## $kick @mention_member
-> Kicks the mentioned member in the server

## $ban @mention_member
-> Bans the mentioned member in the server

## $joke
-> Tells a joke 

## And the bot has a secret command
-> Those who know, know.
